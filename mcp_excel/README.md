MCP Excel Server - Instalador Automático

Este instalador automatiza o clone e a configuração do projeto MCP Excel Server.

- Repositório: https://github.com/yzfly/mcp-excel-server
- Local de instalação: `C:/Projetos/mcp-excel-server`

Pré-requisitos verificados em tempo de execução:
- Git disponível no PATH
- Python instalado
- `uv` instalado (se ausente, o instalador tentará instalar via `pip`)

Como executar manualmente:
- `python mcp_excel/mcp_excel_installer.py`

Através da GUI:
- Selecione “MCP Excel Server” e clique em “Iniciar Instalação”.
